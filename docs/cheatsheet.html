Surgeon AI

Interactive CLI Cheatsheet

Installation & Usage

Run Instantly (Recommended)

Use npx to run the latest version directly without installing it globally.

$npx surgeon-ai scan

Install Globally

Install it globally on your machine to use the `srgn` command anywhere.

$npm install -g surgeon-ai

Core Commands

Initialize Project

Create a configuration file (.surgeon/config.json) in your current project.

$srgn init

Scan Codebase

Run a comprehensive audit on the entire project.

$srgn scan

Review Findings

Open the interactive terminal UI to browse and select issues.

$srgn review

Apply Fixes

Automatically apply fixes for selected issues from the last scan on a new git branch.

$srgn fix --yes

Generate Report

Regenerate Markdown and HTML reports from the last scan data.

$srgn report

Development & Publishing

Publish to NPM

Build the project and publish it to the npm registry.

$pnpm run publisher

Link for Local Testing

Make this package available locally so you can test it in other projects before publishing.

$pnpm link --global
